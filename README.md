# formPHP Dashboard

A form on php

## Requirements & Installation

You can run this project using a local server environment like **XAMPP**, **Laragon**, **WAMP**, or **MAMP**.

### Setup Instructions (XAMPP / Laragon)

1.  **Move the Folder**: Copy your project folder into the server's root directory:
    - For **XAMPP**: `C:\xampp\htdocs\formPHP`
    - For **Laragon**: `C:\laragon\www\formPHP`
2.  **Start Apache**: Open your XAMPP or Laragon control panel and start the **Apache** service.
3.  **Open in Browser**: Visit the URL: [http://localhost/formPHP](http://localhost/formPHP)

## 📁 Project Structure
- `index.php`: Main page content.
- `partials/`: Modular PHP includes (header/footer).
- `styles/main.css`: Premium Vanilla CSS system.
